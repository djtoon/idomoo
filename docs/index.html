01 · Install the CLI

Install on your machine

Pick your OS. Native installers bundle everything — no Node.js required.

terminal 
curl -fsSL https://raw.githubusercontent.com/djtoon/idomoo/main/scripts/install.sh | bash

Installs to ~/.local/bin/idomoo. Supports Apple Silicon and Intel Macs.

terminal 
curl -fsSL https://raw.githubusercontent.com/djtoon/idomoo/main/scripts/install.sh | bash

Installs to ~/.local/bin/idomoo. Supports x86_64 and arm64. Works on WSL.

powershell 
irm https://raw.githubusercontent.com/djtoon/idomoo/main/scripts/install.ps1 | iex

Installs to %LOCALAPPDATA%\Programs\idomoo and adds it to your user PATH automatically.

verify & log in 
idomoo --help
idomoo login   # enter your Account ID & API Secret Key
02 · Add the Agent Skill

Teach your AI agent Idomoo

The skill teaches Claude, Cursor, and other AI agents how to drive the CLI in plain English. Pick whichever install matches your setup.

terminal 
npx skills add djtoon/idomoo

Powered by skills.sh — works across Claude Code, Cursor, GitHub Copilot, Cline, Windsurf, Zed, Aider, and 18+ other AI agents. Requires Node.js ≥ 18.

claude code 
# 1. register the marketplace
/plugin marketplace add djtoon/idomoo

# 2. install the plugin
/plugin install idomoo@idomoo

The fastest path for Claude Code users — marketplace + plugin in two commands. Claude auto-invokes the skill whenever you mention Idomoo or AI video.

terminal 
mkdir -p ~/.claude/skills/idomoo
curl -fsSL https://raw.githubusercontent.com/djtoon/idomoo/main/skills/idomoo/SKILL.md \
  -o ~/.claude/skills/idomoo/SKILL.md

Drops SKILL.md into Claude Code's user skills directory. Works the same for other agents that read user skills from ~/.claude/skills/. Replace the path for agents with different skill dirs (Cursor, Cline, etc).

powershell 
$dir = "$env:USERPROFILE\.claude\skills\idomoo"
New-Item -ItemType Directory -Path $dir -Force | Out-Null
irm https://raw.githubusercontent.com/djtoon/idomoo/main/skills/idomoo/SKILL.md `
    -OutFile "$dir\SKILL.md"

Installs SKILL.md into %USERPROFILE%\.claude\skills\idomoo\. Restart Claude Code to pick it up.
