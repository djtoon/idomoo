Idomoo Agent Skills

Give your AI agent the power to create videos.

Idomoo Agent Skills give AI agents the context and instructions they need to generate AI videos using the Idomoo CLI — covering login, brief creation, blueprints, rendering, and troubleshooting — all on demand.

Skills CLI

Install with npx

Add the Idomoo skill to any supported agent using the skills CLI:

terminal 
npx skills add djtoon/idomoo
Claude Code

Install as a Claude Code plugin

Add the marketplace and install the plugin from inside Claude Code:

claude code 
# Step 1 — register the marketplace
/plugin marketplace add djtoon/idomoo

# Step 2 — install the plugin
/plugin install idomoo@idomoo

Included skill

idomoo

Teaches your agent how to use the idomoo CLI end-to-end: installing the npm package, authenticating with OAuth2, and running the brief → blueprint → ai-video pipeline. Includes all flags, status values, polling behaviour, and troubleshooting steps.

Before you start

Prerequisites

Install the Idomoo CLI once before your agent uses it:

terminal 
npm install -g idomoo-cli

# Authenticate once — prompts for Account ID and API Secret Key
idomoo login
ℹ

Requires Node.js ≥ 18. Your Account ID and API Secret Key are available in the Idomoo dashboard. Credentials are cached at ~/.idomoo/config.json.
